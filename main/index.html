JavaScript Data Types

JavaScript has several built-in types that help you store and manipulate different kinds of values:

Data Type Description Example
String Textual data wrapped in quotes let name = "Israel";
Number Numeric values let age = 25;
Boolean Logical true or false let isAdmin = true;
Array List of items in square brackets let colors = ["red", "blue", "green"];
Object Collection of key-value pairs let user = { name: "Israel", age: 25 };

Comparison Operators

Used to compare two values and return a Boolean result (true or false).

Operator Description Example
== Equal to (value) 5 == "5" // true
=== Strictly equal (type + value) 5 === "5" // false
!= Not equal (loose) 5 != "5" // false
!== Strictly not equal 5 !== "5" // true
> Greater than 10 > 5 // true
< Less than 2 < 5 // true
>= Greater than or equal to 5 >= 5 // true
<= Less than or equal to 3 <= 2 // false

Logical Operators

Used to combine or invert Boolean values.

Operator Name Description Example
&& AND True if both conditions are true true && false // false
|| OR True if at least one condition is true true || false // true
! NOT Inverts the Boolean value !true // false
